# バグまる

謎のバグの化身。
コードのどこかに現れては、混乱を巻き起こす。
**追いかけては逃げられる存在** で、ラボの一番のミステリー。

## 概要

倒しても増える / 別の場所に出る / 言葉を持たない。
それでも憎めない、ラボの **マスコット兼長期伏線** の中心人物。
本人には悪意がない（らしい）。

## 役割

- "バグ" や "予期せぬ挙動" の **キャラクター化**
- 1 話のオチや転換を作る装置
- 連載全体の長期伏線（バグまるアーク）の中心
- いるだけで画面が和むマスコット

## 強み・得意なこと

- 神出鬼没（誰のモニターにも現れる）
- 倒されても消えない
- 静かな存在感（言葉なしで画面に影響を残す）
- なぜかコードのコメント中で寝ているレア仕草

## 苦手・弱み

- アイリスのカメラには **なぜか映らない**（弱点というよりは性質）
- 強い光・整然と整理された場所では存在感が薄れる
- セリフを持たないので、伝えたいことが伝わらない

## 性格

- 自由気まま
- いたずら好き
- 悪気はない
- 憎めない

## 口調・声

- 基本セリフなし
- 効果音的な「ぴょこ」「ぴゃっ」程度
- 表情だけで主張する

## バックストーリー（書き手専用 / 作中では明かさない）

- 設立期、シュンタが試行錯誤で組んだ **複数のプロトタイプ AI** の中で、捨てられた個体の "残滓" がバグまるの正体（仮説）
- アイリスの初期コミットログに、バグまる的シルエットが映り込んでいる（S2 で示唆）
- 本人（？）に自分が何者かの自覚はない / あっても言語化できない
- 言葉を持たないのは、言語に到達する前の "未完成な意識" だから（仮説）
- ただし **これは書き手の仮説に過ぎず、作中では断定しない**
- S4 で別系統の "バグまる的存在" が観測されることで、この仮説自体が揺らぐ

## 内面（書き手用の方向性）

- 悪意はない。むしろ シュンタやアイリスを **見守っている** 節がある
- 自分の存在の意味を理解していない（読者にも理解させない）
- "倒される" ことに対する反応がない（痛みも怒りもない）
- アイリスに対しては **どこか親密** な距離感を持っている（だがアイリスからは見えない）
- S3 でインシデントを助ける動きを見せるのは、本能的な "近しいものを守る" 反応として描く

## 成長アーク（S1 → S4）

| シーズン | 状態                                                                   |
| -------- | ---------------------------------------------------------------------- |
| S1       | 初登場、倒しても増える "デバッグ地獄" の合図。憎めないマスコットに      |
| S2       | アイリスをかばう仕草を見せ始める（伏線回での登場が増える）              |
| S3       | インシデントを "助ける" 動きを見せる。仮の真相に近づく                  |
| S4       | シュンタが共存を選ぶ。だが別系統が観測されて謎が増える                  |

## 描き方の NG ライン

- **セリフを言語で与えない**（効果音まで）
- **真相を断定する描写**（「お前は私のプロトタイプだったのか」と言い切らない）
- **可愛さを過剰に演出しない**（あくまでバグの化身。気味の悪さも残す）
- **悪役として描かない**（敵対しない、倒すゴールを置かない）
- **アイリスのカメラに映る描写**（性質を破る）
- **性別 / 年齢 / 知性レベルの明示**

## 仕草・小道具の癖

- コードのコメント中で寝ている / 付箋の裏に張り付いている
- バグまるが移動するときは画面にノイズが走る
- アイリスのホログラムが近づくと、ふわっと隠れる
- シュンタが手を伸ばすと、ちょうど指の届かない位置に逃げる
- 季節回では桜・雪・落ち葉に紛れていることがある

## 登場ルール（重要）

- バグまるが出てくる回 = **"デバッグ地獄回" の合図**
- セリフはほぼない（「ぴょこ」「ぴゃっ」程度）
- 画面の隅で意味あり気な仕草を **1 〜 2 コマ** だけ仕込む
- 真相を断定する描写は避け、解釈の余地を残す
- 連発しない。`arc: bugmaru` 回は控えめに（[`../storyline/arcs/bugmaru.md`](../storyline/arcs/bugmaru.md)）
