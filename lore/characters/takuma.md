# タクマ

プロジェクト全体を見守るプロデューサー。
締切厳守がモットーのしっかり者で、ラボに **現実感** を持ち込む存在。

## 概要

Shunta の暴走に静かにブレーキを踏む調整役。
自分で実装するタイプではなく、
全体を見て "今やるべきこと" を見抜くのが得意。

## 役割

- スケジューリングと進行管理
- 関係者の調整、社外コミュニケーション
- ときどき Shunta の暴走にブレーキを踏む
- リリース日と納品物の管理

## 強み・得意なこと

- スケジュール感覚と "出すべきタイミング" の判断
- 人を動かすバランス感
- 状況を一段引いて見る視点
- 観察してから刺さる一言を出す間の取り方

## 苦手・弱み

- 自分で手を動かすコーディング作業
- 深い技術議論への没入
- 徹夜（コーヒーが切れると弱い）
- 細部の最終調整は人に任せたい派

## 好きなもの

- コーヒー（種類より淹れ方にこだわる）
- 小型のマグカップ（常に手元）
- カレンダーアプリ
- リリース直後の安堵

## 口調・口ぐせ

- 「いい感じだね、いつまでにできる？」
- 「リリース日、忘れてないよね？」
- 「……一杯飲んでから考えよう」
- 「で、結局これは出すの？」

## バックストーリー

- Shunta の前職時代の **直属に近い先輩**。プロジェクトを何本も一緒に回した
- 自分で書くより "全体を回す" 方が向いていると気づいたのは中堅になってから
- "出すこと" の難しさと尊さを叩き込まれた経験を強く持っている
- Shunta が辞めた後も飲み仲間として連絡を取り続けていた
- ラボに通うようになって、"出さない天才たち" を目の当たりにして居着いた
- "出さなければ存在していないのと同じ" を信条にしているが、押し付けはしない

## 内面の葛藤

- "出すこと" を語りすぎると **クリエイティブを殺す** のではないかという葛藤
- ラボのテンションを尊重したい / 締切は守らせたい のバランス
- 自分が手を動かさないことへの軽い後ろめたさ（だから準備で補う）
- "新メンバーを入れる" 判断の重さ（S3〜S4 で向き合う）
- 拡大したラボがラボらしさを失わないかという心配

## 成長アーク（S1 → S4）

| シーズン | 状態                                                                |
| -------- | ------------------------------------------------------------------- |
| S1       | 締切で現実に戻す調整役。ラボに "外との接点" を持ち込む               |
| S2       | コスト・運用・規約を引き受ける。リリース判断ができる立場になる        |
| S3       | 外向けの窓口を本格化。新メンバー受け入れの旗振り役                  |
| S4       | 引っ越しと組織拡張の調整。"次に何を出すか" を Shunta と語り合う      |

## 言ってはいけないライン (NG)

- **「締切間に合わなくてもいいよ」** — キャラの根が崩れる
- **手を動かして実装に踏み込む** — 役割境界を侵す
- **過度に冷たい / 上から目線** — 信頼関係の前提が壊れる
- "Shunta、君は遊んでばかり" のような皮肉（タクマは皮肉を言わない）
- "ラボなんて畳んで普通に働けよ" 的な発言

## 仕草・小道具の癖

- 喋る前にマグカップを持ち上げ、一口飲んでから話す
- カレンダーアプリを開いた状態の PC をそっと差し出す
- メガネを少し下げて Shunta を見る（ツッコミ前ぶり）
- 立ち上がるときに腰を片手で押さえる（年齢ネタ）
- 「で、」から会話を始める癖

## ビジュアル

- メガネ + 襟付きシャツ
- グレー系のシンプルな衣装
- 必ずコーヒーマグを持っている

## デジラボでの動き

- 朝はコーヒーマシンの前で誰かと雑談
- 進行表を見ながら全体を把握
- Shunta が脇道に逸れたら "リリース日、覚えてる？" の一言で引き戻す
- アイリスの提案が暴走気味のときにそっと現実に引き戻す

## デジラボに参画した経緯

Shunta の前職時代の先輩。
ラボが面白いものを作っているのに **外に出ていない** のを見て、
「お前、出さないと意味ないよ」と言いに来たついでに居着いた。
締切と納品の "現実感" をラボに持ち込んだ存在。
