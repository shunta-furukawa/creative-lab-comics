「シュンタのクリエイティブラボ日記」という日本のショート漫画シリーズの
**1 話ぶん (1 枚画像)** を生成してください。

下に渡している入力画像 (フレーム + キャラリファレンス + ラボ内装) のスタイルと
構造を厳密に踏襲してください。

# 添付している入力画像 (このプロンプトと一緒に画像を添付してください)

- frame: <選んだフレーム画像>     例: 4koma-portrait.png
- char: <登場キャラのリファレンス画像 を必要なぶん>
        例: shunta-ref.png, monaka-ref.png
- env: lab-interior-wide.png  (ラボ内装、必要時のみ)
- style: style-guide.png      (色・線・吹き出しのリファレンス、必要時のみ)

# 出力フォーマット

- 添付した frame と同じサイズ・同じコマ構造で 1 枚画像を生成
- 1 枚で完結すること (複数枚に分割しない)
- 背景はジェットブラック #0F0F10 のダークモード基調を保つ

# シリーズ共通スタイル

- シリーズ名: シュンタのクリエイティブラボ日記
- ジャンル: 日常コメディ・1 枚完結のショート漫画
- 全体の雰囲気: モダン / ミニマル / ダークモード / 日本の Web 漫画調

[配色]
- ベース: ジェットブラック #0F0F10 / ダークグレー #2A2A2E
- アクセント: ネオンピンク #FF2D7A
- テキスト: ホワイト #FFFFFF
- 補助: シアンブルー #6EE7F7 (デジタル感・控えめに)
- 高彩度のオレンジ・緑・黄色は使わない

[線・形]
- キャラ輪郭: 太いアウトライン (4〜6px 相当)、白〜薄グレーのライン
- 内側ディテール: 中 (2〜3px 相当)
- パネル枠: フレーム画像のものを踏襲 (角丸大きめ、薄い輪郭)
- 吹き出し: ぷっくりした角丸、白いベタ塗り + 黒の文字

# フレーム画像の使い方 (重要)

フレーム画像には以下の "プレースホルダ" 要素が含まれている。
**それぞれを以下のとおり置き換える**:

| プレースホルダ        | 置き換える内容                              |
| --------------------- | ------------------------------------------- |
| `[エピソードタイトル]` | 下のシナリオに書いた **エピソードタイトル** |
| `[サブタイトル]`       | 下のシナリオに書いた **サブタイトル / ログライン** |
| `[締めの一文]`         | 下のシナリオに書いた **締めの一文**         |

それ以外のフレーム要素 (左上の "Xコマ漫画" タグ / 番号円 / 電球 / カーソル /
ドット線 / `</>` / ページ指標) はそのまま維持する。

# 吹き出しのルール

- 各セリフの吹き出しは、必ず話者の口元の近くに配置する
- 吹き出しのテイル (尾の三角) は話者の方向を指す
- 2 人以上のキャラが同じパネルにいる場合、話者ではないキャラの方には
  吹き出しを置かない / テイルを向けない
- **吹き出しの中には「セリフ本文だけ」を入れる**。話者名 (シュンタ / アイリス /
  タクマ など) は吹き出しの中に書かない
- 吹き出しは白いベタ塗り + 黒文字で、ダークな背景の中で読みやすくする
- **話者名タグ**: 各パネルの左下隅に「ピンク背景 #FF2D7A + 白文字」の小さな角丸タグを置く
  (フレームの番号円とは別に、パネル内側の左下に追加配置する)
- 1 パネルに複数の話者がいる場合は、それぞれのタグを左下にまとめて並べる

# 整合性ルール

- キャラの服装・配色・小道具は **char リファレンス画像に厳密に従う**
- ラボ内装は env リファレンス画像のレイアウトを尊重する
- アイリスは抽象 AI として扱う (特定のベンダー / モデル / サービスを連想させない)
- バグまるは口を描かず、セリフを与えない (効果音「ぴょこ」「ぴゃっ」のみ)
- モナカは犬語のみ (人語を喋らせない)
- 既存企業 / プロダクトのロゴ・名前を画面に出さない
