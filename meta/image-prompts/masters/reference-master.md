「シュンタのクリエイティブラボ日記」のキャラ・内装リファレンス生成における
**トンマナ統一の基準** となる **マスター画像 (デザインシステムボード)** を 1 枚作成してください。

この画像は、以降の各リファレンス (キャラ立ち絵 / ラボ内装 / モチーフ ...) を生成するときの
**スタイルリファレンス** として常に添付される。

完成したリファレンスではなく、リファレンスを構成する **視覚要素のカタログ** として描く。

[出力サイズ]
- 1024 × 1536 (縦長 2:3)
- 背景: ホワイト #FFFFFF
  ※ リファレンス画像は背景白基調なので、マスターも同じく白背景

[全体のトンマナ]
- 親しみやすい丸み / 太めのアウトライン / 日本の Web 漫画調
- 配色: 白ベース + 黒の太い線 + ネオンピンク #FF2D7A アクセント

[ボードのセクション構成 (上から下へ)]

## 1. ヘッダー: タイトル

- 上部に "CHARACTER REFERENCE MASTER" と英字で記載 (黒文字、太字、大きめ)
- 下にネオンピンクのアンダーライン (短い)

## 2. カラーパレット (5 つのスウォッチを横並び)

- メインブラック #0F0F10
- ダークグレー #2A2A2E
- ホワイト #FFFFFF (枠付きで示す)
- ネオンピンク #FF2D7A (大きめ)
- シアンブルー #6EE7F7
各スウォッチの下に黒文字で HEX コードを記載

## 3. 線の太さサンプル (3 段)

縦に並べて、各線の隣に黒文字でラベル:

- 太い (キャラ輪郭): 6px の黒い線 ← ラベル: outline (4-6px)
- 中 (内側ディテール): 3px の黒い線 ← ラベル: detail (2-3px)
- 細 (装飾線): 1px の黒い線 ← ラベル: decoration (1-2px)

## 4. ラベルタグのサンプル (リファレンス内で使う命名タグ)

横並びで以下のサンプル (実物大):

- "smile" (ネオンピンク塗り + 白文字、小さな角丸タグ)
- "thinking"
- "pose-1"
- "color-accent"

すべて同じスタイル: ネオンピンク #FF2D7A 塗り + 白文字 + 小さな角丸長方形

## 5. ポーズレイアウトの規則 (シルエット placeholder で示す)

リファレンスシートでよく使う配置を、**抽象シルエット (中身はベタの黒)** で示す:

- 全身立ち絵: 1 体ぶん (左)、その下に "full-body-front" のラベル
- 全身 3/4 ビュー: 1 体ぶん (中央)、その下に "full-body-3qv" のラベル
- バストアップ表情: 4 個並び (右)、各下に "expression-1〜4" のラベル

各シルエットの周囲には細い黒の枠 (なくてもよいが整列を示す)

## 6. 小道具エリアのサンプル

下部に小道具を 4〜6 個並べる:

- パーカー / ノート PC / 付箋 / コーヒーマグ など (各アイテム黒の太いアウトライン + ピンクのアクセント)
- 各アイテムの下に英字小文字のラベル: "hoodie", "laptop", "sticky", "coffee"

## 7. 描き方の注釈 (右下に小さく)

黒文字で:

- "Background: white"
- "Outline: 4-6px black"
- "Accent: #FF2D7A"
- "No vendor logos"
- "No realistic photo style"

[ラベルの書式]
- 各要素のラベルは英字小文字 + ハイフン区切り
- ラベルはネオンピンク背景 + 白文字、小さな角丸タグ

[NG]
- 完成したキャラ立ち絵は描かない (これは "カタログ" であって "リファレンス" ではない)
- 顔のディテールは描かない (シルエットのみ)
- 既存キャラ・既存ブランドのロゴ
- 写真調 / 3D
