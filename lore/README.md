迷ったときに立ち戻る "正" の置き場所。

## 🎯 一行で言うと

> **バックエンドからフロントエンド、デザインまで。**
> **コードとアイデアでワクワクを現実に変える。**

主人公 Shunta が、日々のものづくりの中で出会う「ひらめき」「つまずき」「小さな勝利」を、
**1 話 = 画像 1 枚** で切り取って届けるシリーズ。コマ数はネタごとに最適化する。

## 🎨 ビジュアル方針

- ベースは **モノトーン**
- アクセントに **ネオンピンク (#FF2D7A)** で "ワクワク感" を演出
- 補助色に **シアンブルー (#6EE7F7)** を使い、デジタル感・テクノロジー感を表現

詳細は [`../design-system/`](../design-system/) を参照。

## 📐 1 話の作り方

- **1 話 = 画像 1 枚**。コマ数はネタごとに最適化（1 コマ〜可変）
- 1 枚の中で必ず 1 回オチる
- 連作はしてもよいが、各話単独で笑える / 落ちることを最優先

詳しくは [`../design-system/rhythm.md`](../design-system/rhythm.md) を参照。

## 設定を変えるときのルール

- 必ず PR にする（変更履歴を Git に残すのがこのリポジトリの肝）
- "なぜ変えたか" を PR description に書く（未来の自分のため）
- 既存話との矛盾を Claude Code にチェックしてもらう
- 用語の追加・変更は [`./lexicon.md`](./lexicon.md) にも反映する
