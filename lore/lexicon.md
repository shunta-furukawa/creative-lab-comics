## 表記ルール

### 句読点

- 漢字・ひらがなの区切りには **半角スペース** を入れる（"アイリス さん" のような形では入れない）
- 数字は半角、単位は全角・半角混在 OK（例: `4 コマ`、`1 枚`）

### 鍵括弧

- セリフ: 「」
- 二重引用 / 強調: 『』
- 英文・固有名詞: そのまま英字 or ダブルクォート

### キャラ名の英字 vs カナ

- **全員カナで統一**: シュンタ / アイリス / ミカ / タクマ / バグまる / モナカ
- アルファベット表記が必要なら `Shunta / Iris / Mika / Takuma / Bugmaru / Monaka`
- カナと前後の助詞 (が / の / は / に / と / を 等) の **間にスペースは入れない**
  - ✕ `シュンタ が`  ✓ `シュンタが`
  - ✕ `アイリス さん` ✓ `アイリスさん`

### バグまる関連

- 効果音は「ぴょこ」「ぴゃっ」を基本とする
- セリフ（言語化されたもの）は与えない
- 性別・年齢・知性レベルは描写しない

### モナカ関連

- 鳴き声は「わふっ」「くぅーん」「ぱふっ」を基本とする
- セリフは与えない（犬語のみ）
- 種別は **ゴールデンドゥードゥル**。クリーム色のもこもこ
- たまに虚空を見つめる仕草で **バグまるの気配** を匂わせる（断定はしない）

### アイリス関連

- ベンダー / モデル名・サービス名は **絶対に書かない**
- 「○○（外部 AI 名）に聞いた」という台詞も避ける（ぼかすか自前ツールにする）
