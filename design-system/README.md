# デザインシステム

「Shuntaのクリエイティブラボ日記」のビジュアル・トーンを統一するためのルール集。

| ファイル                                       | 内容                                        |
| ---------------------------------------------- | ------------------------------------------- |
| [`color-palette.md`](./color-palette.md)       | カラーパレット                              |
| [`shapes-and-lines.md`](./shapes-and-lines.md) | シェイプ・ライン（角丸 / アウトライン）     |
| [`motifs.md`](./motifs.md)                     | よく出てくるモチーフ                        |
| [`rhythm.md`](./rhythm.md)                     | リズム設計（1 枚で落とす / コマ数自由）      |
| [`speech-bubbles.md`](./speech-bubbles.md)     | 吹き出し / セリフのルール                   |

## 全体方針

- **ベースはモノトーン**、アクセントに **ネオンピンク** で "ワクワク感" を演出
- シアンブルーをサブで使い、デジタル感・テクノロジー感を表現
- 親しみやすい丸みと太い輪郭で、読みやすく、記憶に残るビジュアル
- 日常の小物やデジタル要素で、"つくる人の世界" を表現
- **1 話 = 画像 1 枚**。コマ数は自由、SNS 投稿は 1 枚で完結

## デザインを変えるときのルール

- 設定変更は **必ず PR にする**
- 変更理由を PR description に書く（"なぜそうしたのか" は未来の自分のため）
- 変更後はサンプルを `assets/concept/` に置く
