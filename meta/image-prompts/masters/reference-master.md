**トンマナ統一の基準** となる **マスター画像 (デザインシステムボード)** を 1 枚作成してください。

この画像は、以降の各リファレンス (キャラ立ち絵 / ラボ内装 / モチーフ ...) を生成するときの
**スタイルリファレンス** として常に添付される。

完成したリファレンスではなく、リファレンスを構成する **視覚要素のカタログ** として描く。

[出力サイズ]
- 1024 × 1536 (縦長 2:3)
- 背景: ジェットブラック #0F0F10

[全体のトンマナ]
- ダークモード基調 / モダン / クール / 日本の Web 漫画調
- 配色: 黒ベース + ネオンピンク #FF2D7A + 白 #FFFFFF + 補助シアン #6EE7F7
- 黒背景の上でも **中身は鮮やかに描く**: ピンク・白・カラフルなアクセントで要素を明確に見せる
- 暗い要素 (黒い衣装・黒髪 など) を描くときは、輪郭を白〜薄グレー (#CCCCCC) で縁取って可視性を確保する

[ボードのセクション構成 (上から下へ)]

## 1. ヘッダー: タイトル

- 上部に "CHARACTER REFERENCE MASTER" と英字で記載 (白文字、太字、大きめ)
- 下にネオンピンクのアンダーライン (短い)

## 2. カラーパレット (5 つのスウォッチを横並び)

- メインブラック #0F0F10 (黒の上では薄い枠で囲んで存在を示す)
- ダークグレー #2A2A2E
- ホワイト #FFFFFF (大きめ)
- ネオンピンク #FF2D7A (大きめ)
- シアンブルー #6EE7F7
各スウォッチの下に **白文字で HEX コード** を記載

## 3. 線の太さサンプル (3 段)

縦に並べて、各線の隣に白文字でラベル:

- 太い (キャラ輪郭): 6px の **白い線** ← ラベル: outline (4-6px)
- 中 (内側ディテール): 3px の **白い線** ← ラベル: detail (2-3px)
- 細 (装飾線): 1px の **白い線** ← ラベル: decoration (1-2px)

> 黒背景なのでサンプル線は白で表示する。実際のキャラ立ち絵では、白い肌・白い服など
> 明るい色の要素は **黒〜濃グレー** で輪郭を取る。暗い色の要素は **白〜薄グレー** で取る。

## 4. ラベルタグのサンプル (リファレンス内で使う命名タグ)

横並びで以下のサンプル (実物大):

- "smile" (ネオンピンク #FF2D7A 塗り + 白文字、小さな角丸タグ)
- "thinking"
- "pose-1"
- "color-accent"

すべて同じスタイル: ネオンピンク塗り + 白文字 + 小さな角丸長方形

## 5. ポーズレイアウトの規則 (シルエット placeholder で示す)

リファレンスシートでよく使う配置を、**抽象シルエット (中身は薄グレー #2A2A2E
+ 白い細いアウトライン)** で示す。黒背景でも形が見えるように:

- 全身立ち絵: 1 体ぶん (左)、その下に "full-body-front" のラベル
- 全身 3/4 ビュー: 1 体ぶん (中央)、その下に "full-body-3qv" のラベル
- バストアップ表情: 4 個並び (右)、各下に "expression-1〜4" のラベル

各シルエットの周囲には細い白の枠 (見やすさのため)

## 6. 小道具エリアのサンプル

下部に小道具を 4〜6 個並べる。**カラフルに描く** (黒背景の上で映えるように):

- パーカー (グレー本体 + 白アウトライン + ピンクのフード裏地のアクセント)
- ノート PC (白枠 + シアンブルーの画面 + ピンクのキーアクセント)
- 付箋 (ネオンピンク塗り)
- コーヒーマグ (白本体 + ピンクのハートマーク)
- ペン立て (白本体 + ピンクのペン)
- フラスコ (白の輪郭 + ピンクの液体)

各アイテムの下に **白文字 + ピンクタグ** の英字小文字ラベル:
"hoodie", "laptop", "sticky", "coffee", "pen-cup", "flask"

## 7. 描き方の注釈 (右下に小さく)

白文字で:

- "Background: jet black #0F0F10"
- "Outline: 4-6px (light for dark elements, dark for light elements)"
- "Accent: #FF2D7A"
- "Content stays vibrant - dark bg ≠ dark content"
- "No vendor logos, no realistic photo style"

[ラベルの書式]
- 各要素のラベルは英字小文字 + ハイフン区切り
- ラベルはネオンピンク背景 + 白文字、小さな角丸タグ

[NG]
- 完成したキャラ立ち絵は描かない (これは "カタログ" であって "リファレンス" ではない)
- 顔のディテールは描かない (シルエットのみ)
- 既存キャラ・既存ブランドのロゴ
- 写真調 / 3D
- "暗い・地味" な配色 (黒背景 + 鮮やかなアクセント が原則)
