**マスター画像 (デザインシステムボード)** を 1 枚作成してください。

この画像は、以降の各フレーム (1コマ / 2コマ / 4コマ / 6コマ ...) を生成するときの
**スタイルリファレンス** として常に添付される。

漫画の本編ではなく、フレームを構成する **視覚要素のカタログ** として描く。

[出力サイズ]
- 1024 × 1536 (縦長 2:3)
- 背景: ジェットブラック #0F0F10

[全体のトンマナ]
- ダークモード基調 / モダン / クール / 日本の Web 漫画調
- 配色: 黒ベース + ネオンピンク #FF2D7A + 白 #FFFFFF + ほんのりシアン #6EE7F7

[ボードのセクション構成 (上から下へ)]

## 1. ヘッダー: タイトル

- 上部に "FRAME DESIGN MASTER" と英字で記載 (白文字、太字、大きめ)
- 下にネオンピンクのアンダーライン (短い)

## 2. カラーパレット (4 つのスウォッチを横並び)

- ジェットブラック #0F0F10 (大きめスウォッチ)
- ダークグレー #2A2A2E
- ネオンピンク #FF2D7A
- ホワイト #FFFFFF
各スウォッチの下に小さな白文字で HEX コードを記載

## 3. 共通装飾モチーフのライブラリ (ヘッダー / フッター用)

それぞれを横一列に並べてラベル付きで:

- "</>" コードシンボル (ネオンピンク) ← ラベル: code-symbol
- ピンクのドット線 (短い 5〜7 ドット) ← ラベル: dotted-line
- 電球アイコン (ネオンピンクのライン画 + モーション線 2 本) ← ラベル: lightbulb
- カーソル矢印 (ネオンピンクの塗り) ← ラベル: cursor

## 4. パネル番号円のサンプル (3 サイズ)

横並びで:

- 大: 直径 48px、ネオンピンク塗り + 白の太い数字 "1" ← ラベル: number-large
- 中: 直径 36px、同様の塗り、数字 "1" ← ラベル: number-medium
- 小: 直径 28px、同様の塗り、数字 "1" ← ラベル: number-small

## 5. パネル枠サンプル (1 つの代表例)

- 大きな矩形 (約 800 × 400px)
- 細い薄グレー or くすんだピンクの輪郭 (1〜2px 相当)
- 角丸 24px
- 内側はジェットブラック (空白)
- 左上にネオンピンクの番号円 + 白文字 "1"
- パネルの周囲に矢印付きで以下の注釈を白文字で記載:
  - "thin pink/gray border (1-2px)"
  - "rounded corners 24px"
  - "interior: jet black"
  - "number circle: top-left"

## 6. タイポグラフィサンプル

- "[エピソードタイトル]" (大きい白の太字、角括弧付き) ← ラベル: title
- "[締めの一文]" (小さい白文字、角括弧付き) ← ラベル: closing

[ラベルの書式]
- 各要素のラベルは英字小文字 + ハイフン区切り
- ラベルはネオンピンク背景 + 白文字、小さな角丸タグ
- 各要素の直下、もしくは脇に配置
