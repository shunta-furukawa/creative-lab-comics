# フレーム画像の使い方

フレーム画像のプレースホルダを以下に置き換える:

| プレースホルダ        | 置き換える内容                              |
| --------------------- | ------------------------------------------- |
| `[エピソードタイトル]` | 下のシナリオに書いた **エピソードタイトル** |
| `[締めの一文]`         | 下のシナリオに書いた **締めの一文**         |

それ以外のフレーム要素 (番号円 / 電球 / カーソル / ドット線 / `</>`) はそのまま維持する。

# 吹き出しのルール

- 各セリフの吹き出しは話者の口元の近くに配置する
- テイル (尾の三角) は話者の方向を指す
- 2 人以上のキャラが同じパネルにいる場合、話者ではないキャラの方には吹き出しを置かない
- **吹き出しの中には「セリフ本文だけ」を入れる**。話者名は書かない
- 吹き出しは白いベタ塗り + 黒文字 (ダーク背景上で読みやすく)
- **話者名タグ**: 各パネルの左下隅に「ピンク背景 #FF2D7A + 白文字」の小さな角丸タグ
  (フレームの番号円とは別に、パネル内側の左下に追加配置)
- 1 パネルに複数の話者がいる場合はタグを左下にまとめて並べる

# 整合性ルール

- キャラの服装・配色・小道具は char リファレンス画像に厳密に従う
- ラボ内装は env リファレンス画像のレイアウトを尊重する
- アイリスは抽象 AI として扱う (特定のベンダー / モデル / サービスを連想させない)
- バグまるは口を描かず、セリフを与えない (効果音「ぴょこ」「ぴゃっ」のみ)
- モナカは犬語のみ (人語を喋らせない)
- 既存企業 / プロダクトのロゴ・名前を画面に出さない
