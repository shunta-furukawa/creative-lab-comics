# フレーム画像 生成プロンプト集

> ChatGPT images で **空白のコマ割りフレーム** を生成するためのプロンプト。
> ここで作った画像を `assets/concept/frames/` に保存し、
> エピソード本番生成時に **構造リファレンス（コマ割りの土台）** として添付する。

## 使い方

1. 必要なコマ数 / 配置のセクションを選ぶ
2. **[ベーススタイル]** ([README.md](./README.md) からコピー) を先頭に貼る
3. 続けて **本ファイルの個別プロンプト** を貼る
4. 生成された空白フレームを `assets/concept/frames/<format>.png` として保存
5. エピソード本番生成時、シナリオプロンプトと一緒に **入力画像として添付**

> "フレームの構造をそのまま流用するか / 参考程度にするか" は本番プロンプト側で指示する。
> 厳密に守らせたい場合は「frame の構造に厳密に従う」と明示する。

## 共通仕様（全フレーム共通）

すべてのフレーム画像で共通の仕様:

- **キャンバス**: 縦長の SNS 投稿向け (デフォルト 1024 x 1536 = 2:3)
- **背景**: ホワイト #FFFFFF
- **パネル枠**:
  - 太い黒のアウトライン (6px 相当)
  - 角丸 (大きめ、20〜24px 相当)
  - 内側はホワイト (描画は本番プロンプトで埋める)
- **パネル間ガター**: 細い余白 (16〜24px 程度の白い余白)
- **キャンバス余白**: 上下左右に 40px ほどの白マージン
- **パネル左下**: 各パネルに **キャラ名タグの "予約スポット"** を小さな破線の角丸枠で示す
  - サイズ: パネル全体の 1/8 程度
  - 内側は薄いグレー or 空白
  - 「ここに後でピンクのキャラ名タグが入る」と分かる目印
- **テキストは描かない**: コマには何も描き込まない (空白のフレーム構造のみ)
- **吹き出しも描かない**: 本番プロンプトで配置する

> キャラ名タグの予約スポットを描くのが嫌な場合は、各プロンプトの該当行を削除して OK。
> 本番生成時に AI が自動で配置する場合もある。

## フレームの選び方ガイド

ネタごとにどのフレームを選ぶかの目安:

| ネタの性質                       | おすすめフレーム                               |
| -------------------------------- | ---------------------------------------------- |
| 状況 1 枚で笑わせる              | `1koma-portrait` / `1koma-square`              |
| Before / After / 期待 vs 現実    | `2koma-portrait`                                |
| 短いボケツッコミ                 | `3koma-portrait`                                |
| 起承転結 / 迷ったらこれ          | `4koma-portrait`                                |
| 段取り・手順を見せる             | `6koma-grid`                                    |
| 密度型・テンポで読ませる         | `9koma-grid`                                    |
| 強い 1 コマ + リアクション       | `freeform-a`                                    |
| 比較 → 結論大ゴマ                | `freeform-b`                                    |
| 横長 (ブログ・プレゼン)          | `4koma-landscape`                               |

> リズム設計の詳細は [`../../design-system/rhythm.md`](../../design-system/rhythm.md) を参照。

## 命名規則

`assets/concept/frames/` 配下のファイル名は以下に揃える:

- `<コマ数>koma-<アスペクト>.png`
  - 例: `1koma-portrait.png`, `4koma-portrait.png`, `6koma-grid.png`
- 不定形は `freeform-<a/b/c>.png` で連番
