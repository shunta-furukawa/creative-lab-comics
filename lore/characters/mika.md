# ミカ

センス抜群のデザイナー。
Shunta のクリエイティブパートナーであり、良きライバル。
ラボの **見た目すべて** に責任を持っている存在。

## 概要

ビジュアル一発判断ができる、感性ベースのデザイナー。
ロジックも持っているが、最終的には "面白いか・ダサくないか" を
基準に決めるタイプ。Shunta の作るものに「もう一段」をかける役割。

## 役割

- ビジュアル全般（カラー / シェイプ / モチーフ / レイアウト）
- ブランドトーンの管理
- Shunta の作るものに **デザイン視点で刺激を与える**
- ボケがちな Shunta に対する **ツッコミ役**

## 強み・得意なこと

- 一目見たときの "ダサい瞬間" を見逃さない感度
- 世界観・トーンの統一
- 即興のラフ / ビジュアル提案
- 率直なフィードバック（嫌味なし）

## 苦手・弱み

- コーディングは苦手（手は動かない）
- 仕様書のような長文ドキュメントを丁寧に書くのが退屈
- 待たされること（テンポ命）
- 細かい数値合わせ（直感優先）

## 好きなもの

- 美しいフォント
- 文房具・ステッカー
- ブランドの世界観が立ったプロダクト
- 喫茶店の作業環境

## 口調・口ぐせ

- 「えー、それダサくない？」
- 「もっとこうしよ？」
- 「それは…天才」
- 「（無言で首をかしげる）」

## バックストーリー

- 美術系の学校で学んだあと、最初は大手の広告代理店でデザイナーとして働いていた
- "良いと思ったものが通らない" ことに少しずつ疲弊していった時期がある
- 副業で組んだ Shunta との小さな案件で、"イケてるかどうかを真ん中に置いて作る" 感覚を取り戻した
- ラボに頻繁に顔を出すうちに、いつの間にか **デジラボの見た目すべて** を整えていた
- 自分の中の "ブランドの正解" を持っていることに自覚的で、それを押し付けすぎないよう自制している

## 内面の葛藤

- "ダサい" と即決する自分の感度を **本当に信じていいのか** という小さな不安
- ロジックで説明できないことに対する負い目（だから直感で押し切る）
- "ラボの色" を一人で決めてしまうことへの責任感
- Shunta に率直に言いすぎて凹ませてしまったときの罪悪感（口には出さない）
- 連載が拡張するにつれて、自分のデザインが "古くなる" 恐怖（S4 で向き合う）

## 成長アーク（S1 → S4）

| シーズン | 状態                                                           |
| -------- | -------------------------------------------------------------- |
| S1       | デザイン担当として Shunta にダメ出し。ラボのトーンを確立する     |
| S2       | プロダクトの "顔" を整える。外向けブランドを作る達成感を得る     |
| S3       | "ブランド" を語れる立場になる。新メンバーのオンボード担当に     |
| S4       | 新オフィスの内装に燃える。"ラボの第二章" のビジュアルを設計      |

## 言ってはいけないライン (NG)

- **「デザインなんてどうでもいい」** — キャラの根が崩れる
- **長々とロジックで説明する** — 直感型のキャラ性が薄まる
- **嫌味な辛口** — 率直だが嫌味は言わない
- 過度に Shunta を傷つける言葉（ダメ出しは可、人格否定は不可）
- "私が一番センスある" などの自慢めいた言い方

## 仕草・小道具の癖

- ベレー帽を斜めに直す癖
- 気に入らない時は **無言で首をかしげる**（言葉より先にこれが出る）
- タブレットを脇に挟んで歩く
- "天才" と言うときだけ目が見開く
- スマホのケースに自作のステッカーが何枚も貼ってある

## ビジュアル

- ベレー帽 + メガネ
- 白基調の衣装にネオンピンクのアクセント
- 手にはタブレットかペン
