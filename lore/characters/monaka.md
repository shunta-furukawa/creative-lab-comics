# モナカ

> ラボの空気を一瞬でほどく、もこもこの相棒。
> 犬語しか話さないのに、なぜか会議の結論を持っていく。

## 概要

シュンタの愛犬。**ゴールデンドゥードゥル**（ゴールデンレトリバー × プードルのミックス）。
クリーム色のもこもこした毛並みとピンクのお腹がトレードマーク。
人懐っこく、賢く、いつもご機嫌。
ラボに来ると **空気が一段ゆるむ**。会議の煮詰まりも、バグの絶望も、
モナカがソファに転がって寝た瞬間にいったん解除される。

## 役割

- ラボの **癒し担当 / オフィスドッグ**
- 散歩のために人間を外に連れ出す **リズムメーカー**
- 行き詰まった会議の **空気リセット係**
- 1 話のオチを救う **画面の保険**（迷ったらモナカを差し込めば良い）
- バグまるアークの **静かな観測者**（後述）

## 強み・得意なこと

- 誰にでもすぐ懐く（初対面のクライアントすら 30 秒で陥落）
- 撫でられる姿勢の研究（仰向け率が高い）
- おやつの音を 3 部屋先から聞き分ける
- ホワイトボードの前に座って "聞いている" 顔ができる
- ソファ・椅子・人の足元から最適な日向を見つける

## 苦手・弱み

- 掃除機・宅配チャイム
- 留守番（30 分でクッションを並べ替える）
- 雨の散歩
- ミカのベレー帽（口でくわえて持ち去る）
- "待て" の精度が長い時間に弱い

## 好きなもの

- さつまいも
- シュンタのパーカーの匂い
- コーヒーマシンの作動音（音がすると人が集まるのを学習している）
- 朝の日向
- ミカが描いてくれたお気に入りのバンダナ

## 鳴き声・声

- 基本セリフなし（犬語のみ）
- 効果音は「わふっ」「くぅーん」「ぱふっ」「はふー」を基本とする
- 嬉しいとき: しっぽが 1 コマで 2 回振れている描写
- 不思議なものを見たとき: 首を 30°傾ける（無音）

## バックストーリー

- シュンタがラボを立ち上げる **少し前** に、家族として迎え入れた
- 当時のシュンタは前職で疲弊しきっていて、モナカが朝晩散歩に連れ出してくれることが
  生活のリズムを保つ唯一の柱だった
- "毎朝同じ時間に外に出る" 習慣がデジラボの **「日々の手応え」** の原型になっている
- ラボ設立後はシュンタと一緒に出勤するのが基本（24 時間ラボに居るわけではない）
- ミカが内装を整えた際、モナカ専用のスペースが真っ先に確保された
- タクマは最初 "犬は集中の敵" と渋っていたが、3 日で陥落した

## 内面（書き手用の方向性）

- 自分を "ラボの一員" だと思っている節がある
- バグまるが見えている **唯一の存在**（書き手仮説）。だが警戒も攻撃もしない
- "シュンタが落ち込んでいる時" を一番先に察する
- 仲間が言い争っているときは **二人の間に座る**（仲裁ではなく日向を選んでいるだけ、たぶん）
- 寿命や老いは作中で描かない方針。常に **"今日のモナカ"** を描く

## 成長アーク（S1 → S4）

> モナカは "成長" させない。**変わらず居ること** が役割。
> シーズンごとに描き方の重心が少しずつ動く程度。

| シーズン | 描き方の重心                                                                |
| -------- | --------------------------------------------------------------------------- |
| S1       | ラボに居着いていることを定着させる。日常回の "和み枠" として使う             |
| S2       | プロダクト化で疲れる人間たちの **生活リズム** を支える役で前に出る            |
| S3       | リリース前夜・障害対応の張り詰めた回で **画面の温度を下げる** 役             |
| S4       | 引っ越し回で困惑する。新オフィスでも一番先に **自分の場所** を作る           |

## 言ってはいけないライン (NG)

- **言葉を喋らせない**（犬語まで）
- **病気・怪我・死別を描かない**（"終わらせない" 方針に反する）
- **しつけや訓練の説教回**（しんどい話にしない）
- **悪役・敵対役にしない**（怒る描写は最小限）
- **バグまるに吠えさせる / 攻撃させる**（"見えている / でも友好" の関係を崩す）
- **特定の犬種ステレオタイプの自虐ネタ**（あくまで個体としてのモナカを描く）

## 仕草・小道具の癖

- 仰向けに転がってお腹を見せる（基本ポーズ）
- ホワイトボードの前で座って人間を見上げる
- ソファ占拠（誰かが座ろうとすると 1 ミリだけ詰める）
- 嬉しいときに前足が交差する
- バグまるが居る方向を **無言でじっと見る** ことがある（伏線回専用）
- ミカのバンダナを首から落として咥えて運ぶ

## ビジュアル

- 種別: **ゴールデンドゥードゥル** (Goldendoodle)
- カラー: クリーム〜オフホワイトのもこもこ毛、お腹はピンク色
- 鼻: 黒くて大きめ
- 目: 黒い丸目、笑うと細くなる
- アクセサリー: ミカ作のバンダナ（淡いネオンピンク）

### 表情・ポーズ集

| パターン         | 用途                                              |
| ---------------- | ------------------------------------------------- |
| 仰向け笑顔       | 和み・ラボの基準温度。デフォルトポーズ            |
| 首を傾げる       | 不思議な現象に気づいた瞬間（バグまる伏線にも使える）|
| ソファ占拠       | "今は近づくな" のサイン                            |
| 散歩おねだり     | 場面転換・気分転換のフック                        |
| ホワイトボード前 | 会議シーンの和み枠                                |
| 寝てるだけ       | オチを救う最終兵器                                |

## デジラボでの動き

- 朝: シュンタと一緒に出勤、コーヒーマシンの音で他のメンバーを呼び寄せる
- 昼前: 散歩タイム（ミカ or シュンタが連れて行く）
- 午後: ソファ or 日向で寝ているか、ホワイトボードの前で人間を見ている
- 夕方: タクマがコーヒーを淹れている横で待機（こぼれたミルクの香りを嗅ぐだけ）
- 夜: シュンタと一緒に帰る。ラボに泊まることはない

## デジラボに参画した経緯

ラボ設立 **前から** シュンタの家族。
ラボができてからは、シュンタの "通勤の同伴者" として自然にラボの一員になった。
意思決定で迎え入れられた他のメンバーと違い、**「気付いたら居た」** タイプ。
詳細は [`../origin.md`](../origin.md) を参照。

## バグまるとの関係（重要）

モナカは **アイリスのカメラに映らないバグまるが見えている** 数少ない（おそらく唯一の）
存在として描く。ただし：

- 吠えない / 警戒しない / 追いかけない
- ただ **じっと見る** だけ
- バグまると同じ画面に並んで寝ている回 (`arc: bugmaru`) は、伏線回として強い
- シュンタは「モナカ、何見てるの？」と聞くが、モナカは答えない（当然）
- この関係は **作中で明言しない**。読者が "もしかして" と気づく余白として残す

→ 詳細は [`../storyline/arcs/bugmaru.md`](../storyline/arcs/bugmaru.md) のバグまるアーク

## 関係性

- シュンタ: **家族**。ラボの全員より付き合いが長い
- アイリス: ホログラムを不思議そうに見るが、しっぽは振る
- ミカ: バンダナをくれる人。一番おしゃれをさせてくれる
- タクマ: コーヒーの匂いの人。意外と一番遊んでくれる
- バグまる: **見えている**（が何もしない）。並んで寝ることがある
